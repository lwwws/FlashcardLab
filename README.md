# FlashcardLab 
FlashcardLab is a web application that helps people learn different languages via flashcards using Spaced Repetition System (SRS)
## Built with
* ASP.NET
* Bootstrap
* Microsoft SQL Server
## Javascript libraries used
* jquery
* chart.js
* jvectormap
